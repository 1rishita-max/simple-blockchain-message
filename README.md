# Simple Blockchain Message Storage

This is a beginner-friendly blockchain project.

## Objective
To store and retrieve a message using blockchain smart contracts.

## Technology Used
- Solidity
- Ethereum
- Remix IDE

## How It Works
- A message is stored on the blockchain using a smart contract.
- The stored message can be retrieved anytime.

## Output
Message stored and retrieved successfully using Remix IDE.
